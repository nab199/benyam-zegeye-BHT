# Medical Equipment Images

This directory contains images for the Fresenius 2008K hemodialysis machine and other medical equipment.

## Required Images:

### Fresenius 2008K Hemodialysis Machine
- `fresenius-2008k-main.jpg` - Main image of the complete machine (600x400px recommended)
- `fresenius-2008k-thumb.jpg` - Thumbnail version (100x100px recommended)
- `fresenius-2008k-control.jpg` - Control panel interface (600x400px)
- `fresenius-2008k-control-thumb.jpg` - Control panel thumbnail (100x100px)
- `fresenius-2008k-hydraulic.jpg` - Hydraulic system components (600x400px)
- `fresenius-2008k-hydraulic-thumb.jpg` - Hydraulic system thumbnail (100x100px)

### Other Equipment Images
- `maintenance-tracker.jpg` - Maintenance tracking software interface
- `calibration-system.jpg` - Calibration equipment setup
- `medical-imaging.jpg` - Medical imaging analysis tools
- `prosthetic-control.jpg` - Prosthetic limb control system

## Image Guidelines:
- Use high-quality, professional images
- Ensure proper lighting and focus
- Include equipment in operational settings when possible
- Maintain consistent style and branding
- Optimize for web (JPEG format, appropriate compression)
- Include alt text descriptions for accessibility

## Placeholder Images:
The application includes SVG fallback placeholders that will display if images are not available. These show the equipment name in a styled box format.

## Usage:
Images are referenced in the following components:
- `MedicalEquipmentGallery.jsx` - Main equipment showcase
- `Projects.jsx` - Project thumbnails and featured images
